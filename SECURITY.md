# Security Policy

## Supported Versions

Sicherheitsupdates werden für die aktuell unterstützten Versionen bereitgestellt.

## Meldung von Sicherheitsproblemen

Bitte melde Sicherheitslücken **nicht öffentlich über Issues**.

Stattdessen:

- kontaktiere die Maintainer direkt
- oder sende eine vertrauliche Nachricht

Wir bemühen uns um eine schnelle Bearbeitung.
